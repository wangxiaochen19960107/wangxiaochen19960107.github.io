hexo在github上的构建

By Wxc

2016-05-21 Update date:2016-05-22

第一步

下载并安装Node.js

下载并安装msysgit

第二步

鼠标右键任意地方，选择Git Bash，使用以下命令安装hexo( ps：实际输入命令只需输入$ 后面的命令然后按enter键即可)

$ npm install hexo-cli -g

自己建立一个文件夹存放hexo博客相关数据，例如E:/hexo。

在E:/hexo文件夹右键空白处，选择Git Bash，然后执行以下命令，Hexo会自动在该文件夹下下载搭建网站所需的所有文件:

$ hexo init
$ npm install
$ hexo g
$ hexo s

不关闭控制台，然后用浏览器访问 http://localhost:4000/。此时，你应该看到了一个本地的漂亮的博客了。

第三步

注册Github帐号

创建repository，这个repository名字的格式必须为youname.github.io

部署本地文件到github

编辑E:\hexo下的_config.yml文件，可以使用Notepad++或者sublime等。

在_config.yml最下方，添加如下配置(注意：后面需要有空格):

deploy: 
  type: git
  repository: git@github.com:wangxiaochen19960107/wangxiaochen19960107.github.io.git//注意要写ssh网址，注意不要写http网址。
  branch: master

Github新用户需要设置 ssh，否则上述命令会失败。ssh 的介绍和设置方法请看官方教程 。

$ hexo g
$ hexo d

输入ssh密钥即可把博客上传到Github
